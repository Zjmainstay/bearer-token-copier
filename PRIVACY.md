# 隐私政策

**Bearer Token Auto Copier** 尊重并保护用户隐私。

---

## 1. 数据收集

**本扩展不收集、存储或传输任何用户数据。**

我们不收集：
- 个人身份信息
- 浏览历史记录
- 网站内容
- 用户行为数据
- 身份验证令牌（仅在复制操作时临时存在于内存中）

---

## 2. 数据处理

所有操作均在**您的设备本地**执行：

- **令牌仅临时存在于浏览器内存中**
  - 点击扩展图标时捕获
  - 立即复制到剪贴板
  - 复制操作后从内存释放
  - 浏览器关闭时自动清除

- **无远程服务器**
  - 不向任何服务器发送数据
  - 不使用云服务
  - 完全离线工作

- **无持久化存储**
  - 不保存数据到磁盘
  - 不使用 Cookie 或本地存储
  - 不使用数据库

---

## 3. 第三方共享

**本扩展不与任何第三方共享数据。**

- 无分析工具
- 无广告网络
- 无第三方服务
- 无外部 API

---

## 4. 权限使用

本扩展请求以下权限，仅用于核心功能：

### webRequest
- **用途**：读取 HTTP 请求头以捕获 Authorization 令牌
- **范围**：只读，不修改请求
- **隐私**：不记录完整的请求内容

### activeTab
- **用途**：刷新当前标签页以触发新请求
- **范围**：仅在用户点击扩展图标时使用
- **隐私**：不访问其他标签页

### clipboardWrite
- **用途**：将捕获的令牌复制到剪贴板
- **范围**：只写不读剪贴板
- **隐私**：仅复制 Authorization 令牌

### notifications
- **用途**：向用户显示操作结果
- **范围**：仅操作反馈（成功/失败）
- **隐私**：无营销或广告

### scripting
- **用途**：注入脚本以执行剪贴板操作
- **范围**：仅用于复制功能
- **隐私**：不读取或修改页面内容

**所有权限仅在您主动点击扩展图标时使用。** 无自动后台操作。
